Requests Tutorial

You can use any client to make request. I used Postman for testing.

Home Page

  • Link: /
  • Type: GET
  • Parameter: None
  • Body: None

Create User

  • Link: /users
  • Type: POST
  • Parameter: None
  • Body:
    {
      "firstName": "firstname",
      "lastName": "lastname",
      "age": "age",
      "password": "password"
    }

Get Specific User Info

  • Link: /users
  • Type: GET
  • Parameter: Id
  • Body: None

Login

  • Link: /users
  • Type: POST
  • Parameter: Id
  • Body:
    {
      "password": "password"
    }

Patch User

  • Link: /users
  • Type: PATCH
  • Parameter: Id
  • Body: (as required)
    {
      "firstName": "firstname",
      "lastName": "lastname",
      "age": "age",
      "password": "password"
    }

Delete User

  • Link: /users
  • Type: DELETE
  • Parameter: Id
  • Body: None

Refresh Token

  • Link: /users/refresh
  • Type: POST
  • Parameter: None
  • Body:
    {
      "token": "refresh_token",
    }
